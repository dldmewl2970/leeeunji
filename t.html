설치


JAVA를 사용하기위한 TOOL로 ECLIPSE를 설치하여 보자.

Java를 기반으로 한 통합 개발 환경으로 이클립스 재단이 만들었다. Java는 Sun Microsystems에서 개발한 프로그래밍 언어이므로 태양(sun)과 관련 있는 식(蝕: 예를 들면 개기일식)을 의미하는 이클립스를 이름으로 채택하였다.

아래의 다운로드 링크를 눌러 진행하시면 됩니다. 설치방법은 설치 가이드를 참조하여 주시기 바랍니다.
설치가이드가 어려우신 분들은 설치관련 영상링크를 클릭하여 영상을 참조하여 설치를 진행하여 주시기 바랍니다.


◆ eclipse 다운로드 링크 : eclipse 다운로드


◆ eclipse 설치영상 링크 : 설치영상





설치가이드





다운로드 링크를 클릭 후 접속하면 바로 다운로드 창이 뜹니다.
그러면 사진과 같이 Download 64bit를 클릭하여 줍니다.
(실제 홈페이지에서는 사진의 버젼보다 업데이트가 되어있을 수 있습니다.)






위와 같은 창이 뜨면 Download를 클릭하여 이클립스 설치파일을 다운받아 줍니다.







다음은 Eclipse IDE for Java Developers(자바패키지)를 다운 받아야 하는데
우측상에 windows,Mac,Linux 중에 본인 OS에 맞게 선택하여 다운로드 받아 줍니다.






2가지 모두 다운받아진것을 확인 할 수 있습니다.






eclipse-inst-win64를 실행시킨 후 java developers를 선택한 후
창이 열리면 위와같이 체크 후 install 클릭하여 설치를 진행하여 줍니다.






설치과정에서 중간에 창이 뜨면 위의 빨간 표시된 부분을 체크 한 후
Accept를 클릭하여 계속하여 설치를 진행해 줍니다.






정상적으로 설치완료되고 Launch를 클릭하면 위와 같은 화면이 실행됩니다.






실행 후 위와 같은 창이 뜨는데 workspace를 지정해 줘야 합니다.
원하는 위치에 workspace를 지정해준 후 Launch를 클릭하여 실행하시면 됩니다.






다운로드 받았던 압축파일을 압축 해제하면 eclipse가 있는데
클릭하면 이클립스가 실행됩니다.





API
